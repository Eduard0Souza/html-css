Incluindo Fontes Baixadas

Esse é um modelo que traz as fontes baixadas para satisfazer a vontade do cliente de usar uma fonte própria

Como Fazer?

Basta baixar as fontes desejadas e referenciá-las no CSS usando a regra @font-face.

Capturando uma fonte de um site aleatório

Para captrar uma fonte de um site, basta usar a extensão fontsninja no seu navegador Chorme através do chromewebstore
