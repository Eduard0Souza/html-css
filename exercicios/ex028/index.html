Algumas propriedades de container do GRID

  • display : grid (para que o grid Layout funcione)
  • grid-template-columns (define a quantidade de colunas do grid)
  • grid-template-rows (define a quantidade de linhas do grid)
  • gap (define o espaço entre os itens do grid)
  • align-items (alinhamento vertical do item)
  • justify-items(alinhamento horizontal dos itens)
  • align-content(alinhamento vertical do conteúdo)
  • justify-content(alinhamento horizontal do conteúdo)
  • grid-auto-row (serve para quando você não puder definir o número de linhas do grid, use para criar uma linha padão, por exemplo: grid-auto-row:100px)
  • grid-auto-column( da mesma forma que o grid-auto-row, adiciona uma coluna conforme foi definida)
  • place-item (configura de forma simplificada as propriedades align-items e justify-items)
  • place-content (faz a mesma coisa que o place-item, porém com o content)
  • grid-template ( configura os valores do grid nas linhas e colunas, exemplo: grid-template 100px 100px 100px/ auto auto auto; isso substitui as linhas grid-template-rows e grid-template-columns)
