handshort flex

O Handshort utilizado no felx será sempre na seguinte oredem: grow shrink basis, ficando dessa forma: flex: 0 1 150px; sendo o primeiro item o flex-grow o segundo o flex-shrink e o terceiro o flex-basis

  • flex: 0 1 auto é o mesmo que flex: intial;

  • flex: 0 0 auto é o mesmo que flex:none;

  • o flex: 1 1 auto é o mesmo que flex:auto;
